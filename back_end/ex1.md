1. O que é o Node.js?
   É um ambiente de execução que permite rodar JavaScript fora do navegador, usando o motor V8 do Google Chrome.
2. O Node.js usa qual linguagem de programação?
JavaScript.
3. O Node.js roda no navegador ou fora dele?
Fora do navegador, geralmente no servidor ou no terminal.
4. Para que serve o Node.js?
Serve para criar aplicações de back-end, APIs, servidores, sistemas em tempo real, ferramentas de linha de comando, entre outros.
5. Qual comando é usado para executar um arquivo Node.js no terminal?
node nome_do_arquivo.js
6. O que é o npm?
É o Node Package Manager, o gerenciador de pacotes do Node.js, usado para instalar e gerenciar bibliotecas e dependências.
7. O que significa a palavra módulo no Node.js?
É um arquivo ou conjunto de arquivos que contém código reutilizável, que pode ser importado em outras partes da aplicação.
8. O Node.js é mais usado no front-end ou no back-end?
No back-end.
9. O que é o arquivo package.json?
É um arquivo que guarda informações do projeto, como nome, versão, dependências, scripts e configurações.
10. O Node.js é gratuito ou pago?
É gratuito e de código aberto (open source).
